Which distribution to use ?

We provide severals way to download a Kevoree tools.

  • JavaSE* are compatible with Windows, OSX, Nix* and *BSD system. Android Runtime is compatible to android OS from version 2.3 and later.
  • Java Web Start method aims at maintaining your runtime to the latest updated version. It also a good solution if you want to kickstart a demo ! On windows and osx operating system you only need to double click on the JNLP file for other try the following command in a terminal. 
    $> javaws runtime.jnlp
  • Executable plain jar version is the most compatible solution. You only need to download the Jar file and execute it using the following command in a terminal. 
    $> java -jar runtime.jar
